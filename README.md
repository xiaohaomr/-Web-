# 校园新闻发布系统

## 项目概述
校园新闻发布系统是一个面向校园的新闻管理和发布平台，提供了用户登录、注册、新闻管理、分类管理、评论管理等功能，同时支持管理员和学生两种角色，实现了新闻的发布、编辑、删除、搜索等核心功能。

## 功能模块

### 1. 用户认证模块
- **登录页面** (`index.html`)
  - 支持用户名/邮箱登录
  - 密码加密传输
  - 记住密码功能
  - 跳转到注册页面

- **注册页面** (`Register.html`)
  - 支持学生和管理员两种角色注册
  - 表单验证（用户名、密码、邮箱格式等）
  - 密码强度检查
  - 跳转到登录页面

### 2. 管理员后台模块 (`NewSystem.html`)
- **新闻管理**
  - 发布新闻
  - 编辑新闻
  - 删除新闻
  - 新闻列表展示
  - 新闻搜索功能

- **分类管理**
  - 添加新闻分类
  - 编辑分类名称
  - 删除分类
  - 分类列表展示

- **评论管理**
  - 查看新闻评论
  - 审核评论
  - 删除评论
  - 评论列表展示

- **部门消息** (新增)
  - 查看各部门消息
  - 按部门筛选消息
  - 添加部门消息

- **联系人管理** (新增)
  - 联系人列表展示
  - 搜索联系人
  - 添加联系人

- **新闻回收站** (新增)
  - 查看已删除新闻
  - 恢复已删除新闻
  - 清空回收站

- **系统设置** (新增)
  - 账户设置（修改用户名、密码、邮箱）
  - 系统设置（评论审核、新闻审核、每页显示数量等）
  - 通知设置（系统通知、评论通知、部门消息通知等）

### 3. 学生前台模块 (`studentStyle.html`)
- **首页**
  - 轮播图展示重要新闻
  - 最新新闻列表
  - 热门新闻推荐

- **校园新闻**
  - 新闻分类浏览
  - 新闻详情查看
  - 新闻评论功能

- **论坛**
  - 论坛话题列表
  - 发帖功能
  - 回复功能

- **反馈**
  - 提交反馈意见
  - 查看反馈历史

- **个人资料**
  - 查看个人信息
  - 修改个人资料
  - 查看浏览历史

### 4. 搜索模块 (`search.html`)
- 全局新闻搜索
- 多条件筛选（标题、摘要、学院、内容）
- 搜索结果展示
- 搜索关键词高亮

## 使用说明

### 1. 安装与部署
- 环境要求：支持HTML5、CSS3、JavaScript的现代浏览器
- 服务器要求：支持PHP的Web服务器（如Apache、Nginx）
- 数据库要求：MySQL数据库

- 部署步骤：
  1. 将项目文件上传到Web服务器根目录
  2. 配置MySQL数据库连接（在PHP文件中修改数据库连接信息）
  3. 导入数据库结构（如果提供了SQL文件）
  4. 访问index.html开始使用

### 2. 用户登录
- 访问index.html进入登录页面
- 输入用户名/邮箱和密码
- 点击"登录"按钮

### 3. 用户注册
- 从登录页面点击"注册"链接
- 选择用户角色（学生/管理员）
- 填写注册信息
- 点击"注册"按钮

### 4. 管理员操作
- 登录后进入管理员后台
- 使用侧边导航栏切换不同功能模块
- 发布新闻、管理分类、审核评论等

### 5. 学生操作
- 登录后进入学生前台
- 浏览新闻、参与论坛讨论、提交反馈等

### 6. 新闻搜索
- 在任意页面使用搜索框
- 输入关键词后按回车键或点击搜索按钮
- 在搜索结果页面查看匹配的新闻
